
```js no-beautify
"" + 1 + 0 = "10" // (1)
"" - 1 + 0 = -1 // (2)
true + false = 1
6 / "3" = 2
"2" * "3" = 6
4 + 5 + "px" = "9px"
"$" + 4 + 5 = "$45"
"4" - 2 = 2
"4px" - 2 = NaN
7 / 0 = Infinity
" -9\n" + 5 = " -9\n5"
" -9\n" - 5 = -14
null + 1 = 1 // (3)
undefined + 1 = NaN // (4)
```

1. 文字列 `"" + 1` の加算は `1` を文字列に変換します: `"" + 1 = "1"`, そして `"1" + 0` でも同じルールが適用されます。
2. 減算 `"-"` は(ほとんどの算術演算子のように)数値のみで動作し、空文字列 `""` を `0` に変換します。
3. `null` は数値変換後は `0` になります。
4. `undefined` は数値変換後は `NaN` になります。

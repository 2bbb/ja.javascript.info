# 計算機を作成する

3つのメソッドをもつ `calculator` オブジェクトを作りなさい。:

- `read()` は2つの値を聞き、オブジェクトプロパティとしてそれらを格納します。
- `sum()` は保存した値の合計を返します。
- `mul()` は保存した値を掛け、その結果を返します。

```js
let calculator = {
  // ... your code ...
};

calculator.read();
alert( calculator.sum() );
alert( calculator.mul() );
```

# 名前へのマップ

`user` オブジェクトの配列を持っているとします。それは `user.name` を持っています。それは名前の配列に変換するコードを書いてください。

例:

```js no-beautify
let john = { name: "John", age: 25 };
let pete = { name: "Pete", age: 30 };
let mary = { name: "Mary", age: 28 };

let users = [ john, pete, mary ];

let names = /* ... your code */

alert( names ); // John, Pete, Mary
```

解答: **いいえ、それらは等しくありません**:

違いですが、以下では `f1` でエラーが発生したとき、`.catch` で処理されます:

```js run
promise
  .then(f1)
  .catch(f2);
```

...しかしここでは違います:

```js run
promise
  .then(f1, f2);
```

なぜならエラーはチェーンを下に進み、2番目のコードは `f1` の下のチェーンにはないためです。

つまり、`.then` は次の `.then/catch` へ結果/エラーを渡します。そのため、最初の例では下に `catch` があり、2つ目の例は -- ありません。なので、エラーは処理されません。
